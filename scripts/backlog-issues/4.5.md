# [Phase 4] Verify Function App

## Description

Verify the Function App is running correctly in the production environment after deployment. Test the HTTP endpoint, check logs for errors, and confirm the visitor counter function works. This is the production validation of the .NET 8 / Functions v4 upgrade.

## Acceptance Criteria

- [ ] Function App is in 'Running' state in Azure
- [ ] HTTP trigger endpoint returns 200 OK
- [ ] Visitor counter increments correctly
- [ ] Response format matches frontend expectations
- [ ] No errors in Application Insights or Function App logs
