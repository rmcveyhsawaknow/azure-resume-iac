# [Phase 3] Verify Function App

## Description

Verify the Function App is running correctly in the dev environment after deployment. Test the HTTP endpoint, check logs for errors, and confirm the visitor counter function works. This validates the .NET 8 / Functions v4 upgrade in a live Azure environment.

## Acceptance Criteria

- [ ] Function App is in 'Running' state in Azure
- [ ] HTTP trigger endpoint returns 200 OK
- [ ] Visitor counter increments correctly
- [ ] Response format matches frontend expectations
- [ ] No errors in Application Insights or Function App logs

## Dependencies

- Task 3.5: Verify IaC deployment

N/A — requires Azure Portal/CLI access and HTTP endpoint testing.

## Notes

Use `curl` or Postman to test the Function App endpoint directly before testing through the frontend.
