# [Phase 0] Assess Storage Accounts

## Description

Assess all Azure Storage Accounts in the subscription. Identify which are used for the static frontend ($web container), Function App backing storage, and any that are stale. Document storage account names, types, replication, and blob containers.

## Acceptance Criteria

- [ ] All storage accounts listed with name, kind, replication, and resource group
- [ ] Identified which storage account hosts the static site ($web container)
- [ ] Identified which storage account backs the Function App
- [ ] Flagged any unused/stale storage accounts for Phase 5 cleanup
- [ ] Static website configuration (index/error documents) documented

## Dependencies

- Task 0.1: Verify Azure SP credential

## Copilot Instructions Reference

Copilot can assist with scripting `az storage account list` commands.

## Notes

There may be multiple storage accounts — some may be leftover from previous deployments. Phase 5 will handle cleanup of storage accounts #2 and #3.
