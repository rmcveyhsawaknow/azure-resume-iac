```
Senior Principal Agentic Platform Engineer
Job Category: Engineering-Systems
Requisition Number: AGENT017307
Posted: March 27, 2026
Full-Time
On-site
Location: Indianapolis, IN 46219, USA

V2X is seeking a Senior Principal Agentic Platform Engineer to lead the design and delivery of a multi cloud enterprise infrastructure for agentic (autonomous, goal oriented) AI solutions. You will architect and implement secure, compliant, and scalable platforms across AWS, Azure, and Google Cloud (GCP) that power mission critical systems—integrating LLM orchestration, retrieval augmented generation (RAG), and multi agent workflows with strong observability, automation, and reliability.

In this role, you will:

- Own end‑to‑end platform architecture for agentic workloads (cloud, edge, on‑prem), balancing performance, cost, and compliance.
- Establish Infrastructure as Code (IaC), multi‑cloud patterns, service mesh, and API gateways for repeatable, auditable deployments.
- Implement security‑by‑design aligned to NIST RMF, FedRAMP, CMMC 2.0, STIG, and CIS Benchmarks, applying Zero Trust principles and driving assessment/ATO readiness.
- Integrate cloud‑managed AI services (e.g., Azure OpenAI, Amazon Bedrock, Google Vertex AI) and agent orchestration with RAG pipelines, vector databases, and event streaming.
- Lead cross‑functional teams (data science, software, security, DevOps) to deliver resilient platforms, SLAs/SLOs, and continuous improvements.
- Serve as a technical leader and customer‑facing expert, translating program requirements into measurable platform capabilities and risk‑managed roadmaps.

Required Skills:

- 12+ years in cloud/platform engineering with 5+ years leading complex, production systems; 2+ years building infrastructure for LLM/agentic applications (e.g., prompt routing, model serving, guardrails, evaluation pipelines).
- Demonstrated expertise across AWS, Azure, and GCP: identity and access (IAM/Entra/Cloud IAM), networking, storage, compute (including GPU scheduling), and managed AI services (Azure OpenAI, Amazon Bedrock, Google Vertex AI).
- Container orchestration and multi‑cluster patterns (e.g., Kubernetes); service mesh (e.g., Istio); API gateways; secret management (AWS KMS, Azure Key Vault, GCP KMS).
- IaC and automation (e.g., Terraform/Pulumi); CI/CD (e.g., GitHub Actions/GitLab CI/Azure DevOps); policy‑as‑code (e.g., OPA); configuration management.
- Security and compliance: NIST SP 800‑53, RMF/ATO, CMMC 2.0, FedRAMP, STIG, CIS Benchmarks, vulnerability management, SBOM/software supply chain.
- Knowledge and understanding of ServiceNow for IT management and work tracking.
- Data/AI integration: RAG pipelines, vector stores (e.g. OpenAI Search, Kendra, Vertex AI Search), event streaming (e.g., Kafka, Pub/Sub, EventBridge).
- Leadership: program execution, risk and opportunity management, stakeholder communication, supplier oversight; ability to mentor engineers and lead multi‑disciplinary teams.
- U.S. Citizen with ability to obtain a DoD Secret clearance (if program requires).

Desired Skills:

- Experience with agent frameworks, orchestration patterns, prompt operations, guardrails, and safety evaluators.
- Hands‑on with Azure OpenAI, Amazon Bedrock, Google Vertex AI, and open‑source LLMs (e.g., Llama, Mistral); familiarity with relevant features in Gov/DoD cloud environments.
- Knowledge of graph data models, symbolic reasoning, or cognitive architectures for explainable agent behavior.
- Edge/tactical deployments (e.g., DoD IL2–IL5, disconnected/denied environments), GPU operators/schedulers, and multi‑cloud cost optimization.
- Experience with DoD acquisition artifacts (RFP/SOW/BOE), trade studies/AoA, and supporting ATO through RMF documentation and control evidence.

Education Requirements:

- Bachelor's degree in Computer Science, Electrical/Computer Engineering, or a related STEM field.
- Advanced degree (M.S./Ph.D.) preferred in AI, Systems, or a related discipline.
- Equivalent professional experience in large‑scale cloud/platform or AI systems engineering may substitute for formal degree requirements.

Benefits include the following:

- Healthcare coverage
- Retirement plan
- Life insurance, AD&D, and disability benefits
- Wellness programs
- Paid time off, including holidays
- Learning and Development resources
- Employee assistance resources
```
