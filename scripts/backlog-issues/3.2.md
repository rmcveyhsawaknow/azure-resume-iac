# [Phase 3] Update GitHub Actions syntax

## Description

Update GitHub Actions workflow syntax to current best practices. Upgrade action versions (e.g., `actions/checkout@v4`), fix deprecated syntax, and ensure workflows follow current GitHub Actions conventions. Update both dev and prod workflow files.

## Acceptance Criteria

- [ ] All actions updated to latest stable versions (checkout@v4, setup-dotnet@v4, etc.)
- [ ] Deprecated syntax replaced (e.g., `set-output`, `save-state`)
- [ ] Node.js 16 actions replaced with Node.js 20 equivalents
- [ ] Workflow permissions explicitly defined
- [ ] Both dev and prod workflow files updated

## Dependencies

- Task 3.1: Update dev workflow variables (single domain)

## Copilot Instructions Reference

Copilot can fully assist with workflow YAML modernization.

## Notes

Check the GitHub Actions changelog for deprecated features. Common issues: `set-output` command, Node.js 12/16 actions.
