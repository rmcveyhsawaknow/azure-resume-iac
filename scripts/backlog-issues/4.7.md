# [Phase 4] Verify DNS resolution (single domain only)

## Description

Verify DNS resolution for the single domain ryanmcvey.me and its subdomains. Confirm that `resume.ryanmcvey.me` resolves to the correct Azure Storage endpoint through Cloudflare. Use `dig`, `nslookup`, or online DNS tools to verify resolution.

## Acceptance Criteria

- [ ] `resume.ryanmcvey.me` resolves to the correct IP/CNAME
- [ ] DNS propagation is complete (check from multiple locations)
- [ ] TTL values are appropriate
- [ ] No stale DNS records interfering with resolution
- [ ] Cloudflare proxy is active (if expected)
