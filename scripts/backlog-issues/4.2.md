# [Phase 4] Verify/rotate prod credentials

## Description

Verify that all production environment credentials (Azure SP, Cloudflare token, etc.) are valid and properly stored in the prod GitHub environment secrets. Rotate any credentials that are expired or close to expiration. This is the final credential check before the production deployment.

## Acceptance Criteria

- [ ] Azure SP credential verified for prod environment
- [ ] Cloudflare token verified for prod DNS operations
- [ ] All required secrets present in prod GitHub environment
- [ ] Expired credentials rotated and secrets updated
- [ ] Credential expiration dates documented

## Dependencies

- Task 3.9: End-to-end validation

## Copilot Instructions Reference

N/A — requires credential management and Azure/Cloudflare portal access.

## Notes

Production credentials may be different from dev. Verify each one independently.
