# [Phase 5] Identify old resources (incl. storage accounts #2 and #3)

## Description

Identify all Azure resources that are no longer needed, including old storage accounts (#2 and #3), orphaned resource groups, and any other stale resources. Create a cleanup plan with the list of resources to remove and their dependencies.

## Acceptance Criteria

- [ ] Complete list of resources to remove
- [ ] Old storage accounts #2 and #3 identified
- [ ] Orphaned resource groups flagged
- [ ] Resource dependencies mapped (to avoid breaking active resources)
- [ ] Cleanup plan documented with order of operations

## Dependencies

- Task 4.9: End-to-end production validation

## Copilot Instructions Reference

Copilot can assist with scripting `az resource list` commands to identify resources.

## Notes

Do not delete anything yet — just identify and plan. Actual deletion happens in Task 5.2.
