# [Phase 3] Verify/rotate dev credentials

## Description

Verify that all dev environment credentials (Azure SP, Cloudflare token, etc.) are valid and properly stored in the dev GitHub environment secrets. Rotate any credentials that are expired or close to expiration. This is the final credential check before the dev deployment.

## Acceptance Criteria

- [ ] Azure SP credential verified for dev environment
- [ ] Cloudflare token verified for dev DNS operations
- [ ] All required secrets present in dev GitHub environment
- [ ] Expired credentials rotated and secrets updated
- [ ] Credential expiration dates documented

## Dependencies

- Task 0.1: Verify Azure SP credential
- Task 0.8: Verify Cloudflare token
- Task 0.11: Verify GitHub secrets

## Copilot Instructions Reference

N/A — requires credential management and Azure/Cloudflare portal access.

## Notes

Do not proceed with dev deployment if any credentials are invalid. This blocks all subsequent deployment tasks.
