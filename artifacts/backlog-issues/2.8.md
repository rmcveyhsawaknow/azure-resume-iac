# [Phase 2] Update social links

## Description

Update social media links (LinkedIn, GitHub, Twitter/X, etc.) in the site header, footer, or contact section. Ensure all links are current, correct, and open in new tabs. Remove any social links that are no longer relevant.

## Acceptance Criteria

- [ ] All social links updated with current URLs
- [ ] Links open in new tabs (`target="_blank"`)
- [ ] Appropriate `rel="noopener noreferrer"` attributes added
- [ ] Icons/images for social platforms are current
- [ ] Dead or irrelevant links removed

## Dependencies

- Task 2.1: Design content layout

## Copilot Instructions Reference

Copilot can fully assist with HTML link updates and accessibility attributes.

## Notes

Common social links: LinkedIn, GitHub, Twitter/X, email. Ensure the GitHub link points to the resume repo.
