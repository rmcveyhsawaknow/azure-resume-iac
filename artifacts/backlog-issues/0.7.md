# [Phase 0] Assess App Insights

## Description

Assess Application Insights resource(s). Document the instrumentation key, connected services, and whether telemetry is flowing. Verify the Function App is configured to send telemetry to App Insights.

## Acceptance Criteria

- [ ] App Insights resource name, resource group, and instrumentation key documented
- [ ] Connection to Function App verified
- [ ] Recent telemetry data checked (last 30 days)
- [ ] Alert rules reviewed if any exist
- [ ] Workspace-based vs classic mode identified

## Dependencies

- Task 0.1: Verify Azure SP credential

## Copilot Instructions Reference

Copilot can assist with scripting `az monitor app-insights` commands.

## Notes

App Insights may need migration from classic to workspace-based if it hasn't been updated.
