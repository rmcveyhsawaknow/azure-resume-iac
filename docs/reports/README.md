# Epic Reports

Post-epic burn-down reports generated from the AgentGitOps phase retrospectives (`docs/retrospectives/phase-{0..5}-retrospective.md`). Each report presents the same project data — 6 phases, 103 issues, 70 PRs, 235 commits — tailored for a different audience.

## Reports

| Report | Audience | Focus |
|---|---|---|
| [Executive Summary](epic-burndown-executive.md) | Business leaders, executives | ROI, AI leverage, deliverables, risk, recommendation |
| [Technical Staff](epic-burndown-technical.md) | Engineers, architects, DevOps | Architecture before/after, code metrics, security, deployment detail |
| [Project Management](epic-burndown-project.md) | PMs, scrum masters, delivery leads | Velocity, phase timeline, backlog composition, lessons learned |
| [LinkedIn Post](linkedin-post.md) | Public / professional network | What it is, what it did, how to try it — with talking points for Q&A |
