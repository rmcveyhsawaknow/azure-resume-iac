# [Phase 1] Upgrade Functions version (v3 → v4)

## Description

Upgrade the Azure Functions runtime from v3 to v4. Update the `host.json` configuration, `local.settings.json`, and any Bicep/ARM templates that specify the Functions runtime version. Functions v4 is required for .NET 8 support.

## Acceptance Criteria

- [ ] `host.json` updated for Functions v4 compatibility
- [ ] `local.settings.json` updated with `FUNCTIONS_WORKER_RUNTIME` and version
- [ ] Bicep/ARM templates updated to specify Functions v4 runtime
- [ ] `FUNCTIONS_EXTENSION_VERSION` set to `~4`
- [ ] No v3-specific configuration remains

## Dependencies

- Task 1.2: Upgrade .NET runtime (.NET Core 3.1 → .NET 8)

## Copilot Instructions Reference

Copilot can fully assist with configuration file updates. Reference `.github/copilot-instructions.md` for backend conventions.

## Notes

Functions v3 reached end-of-life on December 13, 2026 (extended), but v4 is strongly recommended for .NET 8.
