# [Phase 0] Check SSL/TLS settings

## Description

Check SSL/TLS configuration for the ryanmcvey.me domain in Cloudflare. Verify the SSL mode (Full, Flexible, Strict), certificate status, and HTTPS redirect settings. Ensure the resume site will be accessible over HTTPS after deployment.

## Acceptance Criteria

- [ ] SSL/TLS mode documented (Full, Full Strict, Flexible, etc.)
- [ ] Edge certificate status verified (active, expiration date)
- [ ] Always Use HTTPS setting checked
- [ ] Minimum TLS version documented
- [ ] Any custom certificates or origin certificates noted

## Dependencies

- Task 0.8: Verify Cloudflare token

## Copilot Instructions Reference

Copilot can assist with Cloudflare API scripting for SSL/TLS settings.

## Notes

Full (Strict) SSL mode is recommended for production. Verify this is correctly configured.
