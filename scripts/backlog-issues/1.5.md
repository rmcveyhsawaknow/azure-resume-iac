# [Phase 1] Update test project

## Description

Update the test project to target .NET 8 and ensure all unit tests pass. Update test framework packages (xUnit, NUnit, MSTest), mock libraries, and test assertions as needed. All existing tests should pass after the upgrade.

## Acceptance Criteria

- [ ] Test project `.csproj` updated to target `net8.0`
- [ ] Test framework packages updated to latest compatible versions
- [ ] All existing unit tests compile and pass
- [ ] Any new tests added for upgraded functionality
- [ ] `dotnet test` succeeds with all tests passing

## Dependencies

- Task 1.2: Upgrade .NET runtime (.NET Core 3.1 → .NET 8)
- Task 1.4: Update NuGet packages

Copilot can fully assist with test project updates and writing new tests. Reference `.github/copilot-instructions.md` for testing conventions.
