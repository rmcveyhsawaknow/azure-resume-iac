# [Phase 1] Verify Key Vault access

## Description

Verify the Function App's managed identity has access to the Key Vault secrets it needs (e.g., Cosmos DB connection string). Test that the Key Vault reference in app settings resolves correctly. Ensure the access model (RBAC or access policies) is correctly configured.

## Acceptance Criteria

- [ ] Function App managed identity confirmed (system or user-assigned)
- [ ] Key Vault access policy or RBAC role grants GET permission to the identity
- [ ] Key Vault reference syntax in app settings is correct
- [ ] Secret is retrievable by the Function App identity
- [ ] No access denied errors in Function App logs

## Dependencies

- Task 0.5: Assess Key Vault

## Copilot Instructions Reference

N/A — requires Azure Portal/CLI access for identity and access verification.

## Notes

Key Vault references use the syntax `@Microsoft.KeyVault(SecretUri=...)` in app settings.
