# [Phase 3] Verify IaC deployment

## Description

Verify the Infrastructure as Code (Bicep/ARM) deployment completed successfully in the dev environment. Check that all Azure resources were created/updated correctly. Compare the deployed resources against the expected resource list.

## Acceptance Criteria

- [ ] Bicep/ARM deployment completed without errors
- [ ] All expected Azure resources exist in dev resource group
- [ ] Resource configurations match the template parameters
- [ ] No drift between template and deployed state
- [ ] Deployment outputs documented

## Dependencies

- Task 3.4: Push to develop branch

## Copilot Instructions Reference

N/A — requires Azure Portal/CLI access to verify deployed resources.

## Notes

Use `az deployment group show` to check deployment status and outputs.
