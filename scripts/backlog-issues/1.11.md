# [Phase 1] Update workflow dotnet version

## Description

Update the GitHub Actions workflow files to use the correct .NET SDK version (8.0.x) in the `setup-dotnet` action. Ensure both build and test steps use the updated version. This is required for CI/CD to work with the upgraded .NET 8 codebase.

## Acceptance Criteria

- [ ] `actions/setup-dotnet` uses `dotnet-version: '8.0.x'`
- [ ] Both dev and prod workflow files updated
- [ ] No references to .NET Core 3.1 or .NET 6 remain in workflows
- [ ] Build and test steps work with the new SDK version
- [ ] Workflow syntax is valid

## Dependencies

- Task 1.2: Upgrade .NET runtime (.NET Core 3.1 → .NET 8)
