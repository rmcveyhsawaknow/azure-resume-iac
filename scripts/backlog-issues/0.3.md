# [Phase 0] Assess Function App

## Description

Assess the current state of the Azure Function App(s). Document the runtime version, .NET version, app settings, connection strings, and deployment status. Identify whether the Function App is running, stopped, or in an error state. Check the Functions runtime version (expected: v3 with .NET Core 3.1, which is end-of-life).

## Acceptance Criteria

- [ ] Function App name, resource group, and region documented
- [ ] Runtime version and .NET version recorded
- [ ] App settings and connection strings inventoried (names only, not values)
- [ ] Current deployment status and last deployment date noted
- [ ] Health check performed and results documented

## Dependencies

- Task 0.1: Verify Azure SP credential

## Copilot Instructions Reference

Copilot can assist with scripting `az functionapp show` and `az functionapp config` commands.

## Notes

The Function App is expected to be on .NET Core 3.1 / Functions v3, which reached end-of-life. This assessment feeds directly into Phase 1 upgrade tasks.
