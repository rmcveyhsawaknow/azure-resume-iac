# [Phase 2] Update profile photo

## Description

Update the profile photo on the resume site. Replace the existing photo with a current professional headshot. Optimize the image for web (appropriate size, format, compression). Ensure the image is responsive and looks good at all viewport sizes.

## Acceptance Criteria

- [ ] New profile photo added to the frontend assets
- [ ] Image is optimized for web (compressed, appropriate dimensions)
- [ ] HTML `img` tag updated with correct path and alt text
- [ ] Image displays correctly on desktop and mobile
- [ ] Old photo file removed (if replaced)

## Dependencies

- Task 2.1: Design content layout

## Copilot Instructions Reference

N/A — requires selecting and providing the actual photo. Copilot can assist with HTML/CSS for display.

## Notes

Recommended image size: 400x400px or similar square format. Use WebP or optimized JPEG for best performance.
