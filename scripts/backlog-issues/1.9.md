# [Phase 1] Update function key in main.js

## Description

Update the function key or API endpoint URL in the frontend `main.js` file to match the deployed Function App. Ensure the API call uses the correct hostname, function name, and authentication key. The frontend JavaScript makes an API call to the Function App to retrieve the visitor counter.

## Acceptance Criteria

- [ ] API endpoint URL in `main.js` updated to correct Function App hostname
- [ ] Function key or code parameter is correct
- [ ] API call uses HTTPS
- [ ] Error handling in the API call is robust
- [ ] Console logging is appropriate for production

## Dependencies

- Task 1.3: Upgrade Functions version (v3 → v4)

## Copilot Instructions Reference

Copilot can fully assist with JavaScript API endpoint updates.

## Notes

The function key may change after redeployment. Consider using a Function App level key for simplicity.
