# [Phase 0] Inventory resource groups

## Description

List and document all Azure resource groups in the subscription. Identify which resource groups are actively used for the resume project (dev and prod), and flag any that appear stale or orphaned. This provides the baseline for understanding the current infrastructure footprint.

## Acceptance Criteria

- [ ] Complete list of resource groups with location, tags, and resource counts
- [ ] Identified dev vs prod resource groups
- [ ] Flagged any stale/orphaned resource groups for cleanup in Phase 5
- [ ] Documented findings in assessment notes

## Dependencies

- Task 0.1: Verify Azure SP credential

## Copilot Instructions Reference

Copilot can assist with scripting `az group list` commands and formatting output.

## Notes

Look for naming patterns like `*-dev-*` and `*-prod-*` to distinguish environments.
