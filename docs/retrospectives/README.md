## How They're Generated

Each retrospective is generated by [`bootstrap/generate-phase-retrospective.sh`](../../bootstrap/generate-phase-retrospective.sh):

```bash
# Dry run (preview only)
./bootstrap/generate-phase-retrospective.sh 0 --dry-run

# Generate and post to retrospective issue
./bootstrap/generate-phase-retrospective.sh 0
```

The script collects milestone issues, PR/commit stats, Human vs Copilot KPIs, and gap analysis findings, then writes the report here and optionally posts it as a comment on the matching retrospective GitHub issue.

## Workflow

1. Assign phase issues to their milestone
2. Run the generator script at phase end
3. Commit the generated `.md` file
4. Post the report to the retrospective issue
5. Close the milestone
