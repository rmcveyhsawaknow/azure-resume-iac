# [Phase 3] Seed Cosmos DB

## Description

Seed the dev Cosmos DB with initial data if the visitor counter document doesn't exist. Ensure the document has the correct structure, partition key, and initial counter value. If the document already exists from a previous deployment, verify its format is correct.

## Acceptance Criteria

- [ ] Visitor counter document exists in dev Cosmos DB
- [ ] Document has correct `id` and partition key
- [ ] Counter value is a valid integer
- [ ] Document is readable by the Function App
- [ ] Seed script documented for future use

## Dependencies

- Task 3.5: Verify IaC deployment

## Copilot Instructions Reference

Copilot can assist with writing a Cosmos DB seed script.

## Notes

If the container is new, the initial counter value should be set to 0 or the last known production value.
