# [Phase 0] Assess Key Vault

## Description

Assess the Azure Key Vault(s) in use. Document stored secrets (names only), access policies, and which services reference the Key Vault. Verify the Function App has appropriate access to retrieve secrets.

## Acceptance Criteria

- [ ] Key Vault name, resource group, and SKU documented
- [ ] List of secret names (not values) recorded
- [ ] Access policies reviewed — Function App identity has GET permission
- [ ] No expired or disabled secrets that are still referenced
- [ ] RBAC vs access policy model identified

## Dependencies

- Task 0.1: Verify Azure SP credential

## Copilot Instructions Reference

Copilot can assist with scripting `az keyvault` commands for inventory.

## Notes

Key Vault likely stores the Cosmos DB connection string referenced by the Function App.
