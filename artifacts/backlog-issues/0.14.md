# [Phase 0] Phase 0 Retrospective — Assessment

## Description

Wrap-up retrospective for Phase 0 (Assessment). The PM runs the retrospective generator script in a Codespace, reviews the output, commits the summary, and posts the full report to this issue.

## Retrospective Sections

The generator script will produce the following sections:

1. **Phase Summary** — scope, workstreams, and outcomes
2. **Planned vs Actual Effort** — issues planned/closed, PRs merged, commits
3. **Capacity & Duration Metrics** — time, contributors, branches
4. **Human vs Copilot AI Productivity KPI** — task-level and commit-level attribution
5. **Gap Analysis Summary** — findings count, issues created from gap analysis
6. **Next Phase Readiness Checklist**

- [ ] `scripts/generate-phase-retrospective.sh 0` runs without errors
