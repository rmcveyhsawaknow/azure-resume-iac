# [Phase 1] Update NuGet packages

## Description

Update all NuGet package references to versions compatible with .NET 8 and Azure Functions v4. This includes the Azure Functions SDK, Cosmos DB SDK, and any other dependencies. Resolve any package version conflicts and ensure all packages are from supported sources.

## Acceptance Criteria

- [ ] All NuGet packages updated to .NET 8 compatible versions
- [ ] Azure Functions SDK updated to latest v4-compatible version
- [ ] Cosmos DB SDK updated (v3 → v4 if applicable)
- [ ] No deprecated or vulnerable packages remain
- [ ] `dotnet restore` succeeds without warnings

## Dependencies

- Task 1.2: Upgrade .NET runtime (.NET Core 3.1 → .NET 8)
- Task 1.3: Upgrade Functions version (v3 → v4)

## Copilot Instructions Reference

Copilot can fully assist with `.csproj` package reference updates.

## Notes

Check for breaking changes in the Cosmos DB SDK if upgrading from v3 to v4 — the API surface changed significantly.
