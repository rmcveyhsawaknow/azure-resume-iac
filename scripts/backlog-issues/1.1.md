# [Phase 1] Diagnose root cause

## Description

Diagnose the root cause of the Function App failure. Review application logs, App Insights telemetry, deployment history, and runtime configuration to determine why the Function App is not working. This is the critical first step before applying any fixes.

## Acceptance Criteria

- [ ] Root cause identified and documented
- [ ] Application logs reviewed from last 30 days
- [ ] App Insights exceptions and failures analyzed
- [ ] Runtime compatibility issues identified (e.g., .NET Core 3.1 EOL)
- [ ] Clear action plan for remediation defined

## Dependencies

- Task 0.3: Assess Function App
- Task 0.4: Assess Cosmos DB
- Task 0.5: Assess Key Vault

## Copilot Instructions Reference

Copilot can assist with analyzing log patterns and suggesting diagnostic queries.

## Notes

Most likely root cause is .NET Core 3.1 end-of-life causing runtime failures. The Azure Functions v3 runtime is also deprecated.
