# [Phase 5] Remove old resource groups

## Description

Remove old and orphaned Azure resource groups identified in Task 5.1. This includes removing old storage accounts, unused Function Apps, and any other stale resources. Proceed carefully — deleted resources cannot be recovered.

## Acceptance Criteria

- [ ] All identified old resource groups deleted
- [ ] Old storage accounts #2 and #3 removed
- [ ] No active resources accidentally deleted
- [ ] Deletion confirmed via Azure Portal/CLI
- [ ] Cost impact verified (reduced monthly spend)

## Dependencies

- Task 5.1: Identify old resources (incl. storage accounts #2 and #3)

## Copilot Instructions Reference

N/A — requires Azure Portal/CLI access and careful manual execution.

## Notes

Take a final backup or snapshot before deleting if there's any uncertainty. Use `az group delete --no-wait` for non-blocking deletion.
