# [Phase 4] Verify Cloudflare proxy

## Description

Verify Cloudflare proxy is correctly configured for `resume.ryanmcvey.me`. Check that traffic is routed through Cloudflare (orange cloud), caching is working, and security features are active. Verify response headers indicate Cloudflare is in the request path.

## Acceptance Criteria

- [ ] Response headers include `cf-ray` and `server: cloudflare`
- [ ] SSL is terminated at Cloudflare edge
- [ ] Caching headers are appropriate
- [ ] Page rules or cache rules are configured correctly
- [ ] WAF/security features are active
