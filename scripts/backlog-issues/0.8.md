# [Phase 0] Verify Cloudflare token

## Description

Verify the Cloudflare API token stored in GitHub Secrets is valid and has the necessary permissions for DNS management on the ryanmcvey.me zone. This is required for any DNS-related deployment or verification tasks.

## Acceptance Criteria

- [ ] Cloudflare API token retrieved from GitHub Secrets
- [ ] Token verified with Cloudflare API (`/user/tokens/verify`)
- [ ] Token permissions confirmed (DNS edit on ryanmcvey.me zone)
- [ ] Token expiration date documented (if applicable)
- [ ] If invalid, new token generated and GitHub Secret updated

## Dependencies

None — this is a root task.

## Copilot Instructions Reference

N/A — requires Cloudflare dashboard access and credential management.

## Notes

If the Cloudflare token is invalid, DNS and CDN tasks in Phases 3–5 will be blocked.
